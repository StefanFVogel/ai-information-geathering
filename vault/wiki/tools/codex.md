# Codex

OpenAIs CLI/Agent-Tool, Alternative zu [[Claude Code]]. [[Graphify]] unterstützt Codex explizit — User können mit entsprechendem Kommando statt `graphify claude install` den Codex-Hook einrichten, um den [[Knowledge Graph]] auch dort bei jedem Session-Start zu laden.
