# Anthropic

AI-Forschungsunternehmen, Hersteller von Claude (inkl. [[Opus 4.6]], [[Opus 4.7]], Claude Mythus), [[Claude Code]] und der Claude Desktop App. Gegründet 2021 von ehemaligen OpenAI-Forschern.

## Relevanz für diesen Vault

- Model-Release-Zyklus prägt das gesamte AI-Agentic-Ökosystem
- Silent-Parameter-Änderungen (z.B. [[Adaptive Thinking]] + Effort-Default) haben direkte Auswirkungen auf Produktionssysteme
- Claude Code + [[Claude Code Routines]] sind Infrastruktur für autonome Workflows
