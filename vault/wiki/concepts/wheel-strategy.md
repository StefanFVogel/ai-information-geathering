# Wheel Strategy

Options-Strategie zum kontinuierlichen Generieren von Prämien-Einkommen mit einer einzelnen Long-Position. Man verkauft zuerst Cash-Secured Puts. Wird man angedient, schreibt man auf die gekauften Shares Covered Calls — wird man angerufen, beginnt das Rad von vorn. Ziel: solange Prämien einsammeln, bis die Aktie "weggeht".

## Voraussetzungen

- Options-Approval beim Broker
- Cash-Reserve oder Long-Position im Underlying
- Ein Stock, bei dem man das langfristige Halten akzeptiert

## Einsatz in [[AI Trading]]

[[Samin Yasar]] erwähnt die Wheel Strategy als Plan für Folge-Experimente nach der OpenClaw-Challenge — idee: auf einen Stock fixieren und das Rad von einem Agent laufen lassen.
