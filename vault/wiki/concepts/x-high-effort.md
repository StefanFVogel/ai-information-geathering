# X-High Effort

Neues Effort-Level in [[Opus 4.7]], oberhalb von "max". Exklusiv für 4.7 (nicht rückwirkend für [[Opus 4.6]] verfügbar). Löst das vorher kritisierte Shallow-Thinking in hochkomplexen Coding-Tasks.

## Einordnung

Effort-Hierarchie in Opus 4.7: `low < medium < high < max < x-high`. Bei [[Adaptive Thinking]] entscheidet das Modell selbst, wann welches Level greift — X-High wird typischerweise bei harten, ambigen Tasks genutzt.
